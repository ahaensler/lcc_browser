Introduction

This is an embedded web browser. It is showing the local file "index.html". You can use it for example to create a control panel for your LCC setup. The application provides an LCC interface in Javascript.

Examples

LCC events are sent like this.

<button onClick="LCC.sendRawEvent('0123456789AB.0000');">
Send event
</button>

Try it:

LCC events are received like this.

LCC.addEventListener("raw-event", event => {
    const eventId = event.detail;
    const nodeId = eventId.slice(0, 2*6);
    if (nodeId === "104265632192") {
        alert("Received Event!");
    }
})

Dev tools for debugging are available in the context menu.
